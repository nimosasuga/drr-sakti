## 🎯 Use Cases

### Use Case 1: **Saya Admin baru, mau lihat statistik PM**
```
Step 1: Baca QUICK_START_ADMIN_DASHBOARD.md
Step 2: Login ke aplikasi sebagai Super Admin/Koordinator
Step 3: Swipe dari kiri → Pilih \"Admin Dashboard\"
Step 4: Lihat Tab Statistik
```

### Use Case 2: **Saya Developer, mau implement dashboard**
```
Step 1: Baca STRUKTUR_PROJECT_COMPLETE.md (pahami arsitektur)
Step 2: Baca INTEGRATION_GUIDE.md (setup steps)
Step 3: Copy files sesuai panduan
Step 4: Run `flutter clean && flutter pub get`
Step 5: Test di device
```

### Use Case 3: **Saya QA, mau test fitur baru**
```
Step 1: Baca QUICK_START_ADMIN_DASHBOARD.md (pahami fitur)
Step 2: Baca ADMIN_DASHBOARD_README.md (detail behavior)
Step 3: Lihat Testing Checklist di INTEGRATION_GUIDE.md
Step 4: Execute test cases
Step 5: Report bugs if any
```

### Use Case 4: **Saya butuh download data unit dalam Excel**
```
Step 1: Login sebagai Super Admin/Koordinator
Step 2: Buka Admin Dashboard dari sidebar
Step 3: Pilih Tab \"Download\"
Step 4: Tap tombol \"Download Excel\"
Step 5: Grant permission jika diminta
Step 6: File tersimpan di Downloads folder
```
